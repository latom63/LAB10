Питання: Як у Python створити список та додати до нього елементи?

Студент 2: Петренко
Відповідь: У Python список створюється за допомогою квадратних дужок []. Для додавання елементів використовують метод append().
Питання: Що таке декоратор у Python і як його використовувати?

Студент 3: Сидоренко
Відповідь: Декоратор у Python — це функція, яка змінює поведінку іншої функції або методу. Використовується за допомогою синтаксису @decorator_name над функцією.
